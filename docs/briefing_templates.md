## Overview

Briefings are automated daily and weekly summaries personalized to each founder. They aggregate data from:
- KPI metrics (Granola, ZeroBooks)
- Meetings (Zoom, Fireflies, Otter)
- Communications (Slack, Discord, Email)
- Tasks (Monday, Notion)
- Insights and recommendations

### Briefing Types

| Type | Frequency | Default Time | Sections |
|------|-----------|--------------|----------|
| **Morning Brief** | Daily | 7:00 AM local | Calendar, KPIs, Urgent Items, Tasks Due |
| **Evening Wrap** | Daily | 6:00 PM local | Accomplishments, Insights, Tomorrow Preview |
| **Investor Summary** | Weekly | Friday 6:00 PM | Metrics, Highlights, Challenges, Milestones |

### Design Principles

1. **Concise:** Maximum 2-3 minutes to read
2. **Actionable:** Focus on items requiring attention
3. **Contextualized:** Include relevant background
4. **Prioritized:** Most important items first
5. **Consistent:** Predictable structure
6. **Personalized:** Adapt to founder preferences

#### 1. Personalized Greeting
```python
def generate_greeting(founder, context):
    """
    Generate contextual greeting based on:
    - Time of day
    - Recent activity
    - Upcoming events
    """
    greetings = []

    # Time-based
    hour = datetime.now().hour
    if hour < 12:
        base = "Hope you slept well!"
    elif hour < 17:
        base = "Hope your day is going great!"
    else:
        base = "Hope you had a productive day!"

    greetings.append(base)

    # Context-based
    if context.get('big_meeting_today'):
        greetings.append(f"Big {context['meeting_type']} meeting with {context['attendees']} today.")

    if context.get('milestone_achieved'):
        greetings.append(f"Congrats on {context['milestone']}!")

    return " ".join(greetings)
```

        SELECT
            c.subject,
            c.snippet,

            c.platform,
            c.urgency,
            c.received_at,
            c.sentiment

async def format_accomplishments(workspace_id, founder_id, date):
    # Get completed tasks
    completed = await db.fetch("""
        SELECT COUNT(*)
        FROM work.tasks
        WHERE workspace_id = $1
          AND founder_id = $2
          AND DATE(completed_at) = $3
    """, workspace_id, founder_id, date)

    # Get meetings attended
    meetings = await db.fetch("""
        SELECT COUNT(*)
        FROM meetings.meetings
        WHERE workspace_id = $1
          AND founder_id = $2
          AND DATE(start_time) = $3
          AND start_time < now()
    """, workspace_id, founder_id, date)

    # Get key tasks
    highlights = await db.fetch("""
        SELECT title, priority, metadata
        FROM work.tasks
        WHERE workspace_id = $1
          AND founder_id = $2
          AND DATE(completed_at) = $3
          AND (priority IN ('high', 'urgent') OR tags && ARRAY['milestone'])
        ORDER BY completed_at DESC
        LIMIT 5
    """, workspace_id, founder_id, date)

    # Format
    output = f"**Tasks Completed:** {completed['count']}\n"
    output += f"**Meetings Attended:** {meetings['count']}\n\n"

    if highlights:
        output += "**Highlights:**\n"
        for task in highlights:
            output += f"- ‚úÖ {task['title']}\n"

    return output

### Adaptation Logic

```python
def personalize_briefing(template, founder_preferences, data):
    """
    Personalize briefing based on founder preferences
    """
    # Apply tone
    if founder_preferences.get('briefing_tone') == 'concise':
        template = condense_template(template)
    elif founder_preferences.get('briefing_tone') == 'casual':
        template = casualize_language(template)

    # Filter KPIs
    included_kpis = founder_preferences.get('include_kpis', [])
    data['kpis'] = [k for k in data['kpis'] if k['name'] in included_kpis]

    # Apply insight confidence threshold
    min_confidence = founder_preferences.get('insight_minimum_confidence', 0.7)
    data['insights'] = [
        i for i in data['insights']
        if i['confidence'] >= min_confidence
    ]

    # Remove emojis if disabled
    if not founder_preferences.get('include_emojis', True):
        template = remove_emojis(template)

    return template
```

## Formatting Guidelines

### Length Targets
- **Morning Brief:** 300-500 words (~2 min read)
- **Evening Wrap:** 250-400 words (~1.5 min read)
- **Investor Summary:** 400-600 words (~3 min read)

### Voice & Tone
- **Active voice:** "MRR grew 8%" not "MRR was grown by 8%"
- **Specific:** "5 new customers" not "several customers"
- **Confident:** State facts directly
- **Honest:** Don't hide challenges
- **Forward-looking:** Always include next steps

### Data Presentation
- Use **bold** for metrics and important items
- Use emojis sparingly (1-2 per section)
- Always include units (%, $, count)
- Show trends with arrows: ‚ÜóÔ∏è ‚ÜòÔ∏è ‚Üí
- Use tables for metric comparisons

### Priority Indicators
```
üî¥ Critical - requires immediate attention
üü† High - important but not blocking
üü° Medium - should address soon
üü¢ Low - informational
```
